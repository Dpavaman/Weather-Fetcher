### Weather Fetcher App
`This App is developed using HTML, JavaScript and Bootstrap framework`

#### Powered by **REST Countries API** and **Weather API**

1. Information of all the countries is fetched using REST countries API 
2. Information of every individual coutry such as lattitude, longitude, city Name are used to fetch the live weather status of individual countries
3. Bootstrap framework is used for displaying country flag, lattitude, longityde and a fetch weather button within a bootstrap card element
